Windows下nvm,node,npm安装和使用

windows下nvm，node，npm的安装配置

一、介绍

node.js

  • Node.js是一个能够在服务器端运行JavaScript的开放源代码、跨平台JavaScript运行环境。
  • Node.js的出现使JavaScript也能用于服务器端编程。Node.js含有一系列内置模块，使得程序可以脱离Apache HTTP Server或IIS，作为独立服务器运行。

npm

  • npm（全称Node Package Manager，即node包管理器）
  • 是Node.js默认的、以JavaScript编写的软件包管理系统,会随着Node.js自动安装。

nvm

  • nvm，是用来管理不同版本的node.js,实现自由切换。

三、使用NVM(Mac 和 Windows 均适用)

查看当前版本

nvm v
nvm version

安装node.js

nvm install latest

ps: 这里是安装最新版本的node.js 所以后面加的是latest

如果想要安装指定版本的node.js 那么将 ‘latest’ 替换为相应版本号即可

例如：

nvm install 8.9.1

查看已经安装的node.js版本

nvm ls

切换node.js版本

nvm use 8.9.1

列出所有在服务器上可获取的node.js版本

nvm ls available
