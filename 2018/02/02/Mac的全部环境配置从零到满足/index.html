Mac的全部环境配置从零到满足

发表于 2018-02-02 | 分类于 教程 ， Mac

Mac的全部环境装到你欲仙欲死

准备好一切的软件，工具等

  1. JDK 1.8 http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

  2. Mysql 5.7.21 https://dev.mysql.com/downloads/file/?id=475582

    注：一定要选择dmg的那个版本

  3. Maven 3.5.2 http://maven.apache.org/download.cgi

  4. Tomcat 8和8.5 https://tomcat.apache.org/download-80.cgi

  5. ShodowsocksX-NG https://github.com/shadowsocks/ShadowsocksX-NG/releases/

  6. Eclipse EE https://www.eclipse.org/downloads/download.php?file=/oomph/epp/oxygen/R2/eclipse-inst-mac64.tar.gz

    注：下载的只是eclipse的安装器，完成后打开会提示给你一个版本选择下载安装。

  7. Sougou mac https://pinyin.sogou.com/mac/

  8. Sublime text 3 http://www.sdifen.com/sublimetext3.html

  9. Chrome http://www.google.cn/chrome/browser/desktop/

    注：谷歌浏览器就下载百度搜索出来的，官网的太正宗！

开始重装系统！

  1. 备份好数据

  2. 关机状态，按住option+command+R再按一下开机键

  3. 进入选择最后一个磁盘工具

  4. 看清楚，抹除上面的那个磁盘，名字为Macintosh HD ，完成后关掉返回刚刚那个界面

  5. 选择第二个重新安装os，然后傻瓜化操作

  6. 允许任何源 ： sudo spctl --master-disable

  7. 显示隐藏文件 ： defaults write com.apple.finder AppleShowAllFiles -boolean true ; killall Finder

  8. 不显示隐藏文件 ：defaults write com.apple.finder AppleShowAllFiles -boolean false ; killall Finder

  9. 打开系统偏好设置

    语言与地区 ：设置24小时制

    键盘快捷键 ：把切换输入法设置成 command + space 把截图切换成shift+command+space

    触控板 ： 轻点来点按 勾选

    鼠标 ：滚动方向自然 取消勾选

安装环境

  1. JDK 双击正常安装

  2. Sublime 正常安装 (再安装package install ,emmet)

  3. Tomcat 解压到 ~/envir/tomcat

  4. Maven 解压到 ~/envir/maven 并且创建本地仓库

    • 配置请见我的另外一篇博客 mac环境下配置maven ssm项目<>

  5. 安装brew

  6. 安装iterm2

    • 下载地址：https://www.iterm2.com/

    • 设置窗口透明度：

      Preferences -> Profiles -> Window -> Window Appearance 设置 Transparency

  7. 安装zsh

    • sh -c “$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)”

    • 配置：

      1. vim ~/.zshrc先按字母键大写 G（跳至文件末尾），再按下字母键小写 o

        输入 source ~/.bash_profile，最后保存退出。

      2. source ~/.zshrc

  8. 安装node.js

    • brew install node

    • Npm配置淘宝镜像：

      1. vim ~/.npmrc
      2. 先按i进入编辑模式，然后输入：
        registry=https://registry.npm.taobao.org
      3. source ~/.zshrc

  9. 安装git

    • brew install git
    • git config --global user.name "Sotyoyo"
    • git config --global user.email "Sotyoyo@icloud.com"

  10. 安装ssh

    • brew install ssh-copy-id
    • ssh-keygen -t rsa -C "Sotyoyo@icloud.com"

  11. npm安装一些工具(全局)

    • npm install -g hexo-cli
    • npm install -g vue-cli

  12. mysql安装(5.7版本)

    • 正常安装
    • 弹出一个警告框，复制下面信息(每个人的信息不一样)
      2018-01-31T08:39:54.742853Z 1 [Note]
      A temporary password is generated for root@localhost: ioroD7Y=xDP&
      最后面的就是你的密码
    • 修改mysql 的 root@localhost 密码。
      1. 打开系统设置，下面有一个mysql 保证mysql为running状态。
      2. touch ~/.bash_profile
      3. sudo vim ~/.bash_profile
        加一行export PATH=${PATH}:/usr/local/mysql/bin保存退出
      4. source ~/.bash_profile
      5. mysql -uroot -p ,然后将安装的时候保存的随机密码粘贴进去回车。
      6. SET PASSWORD FOR ‘root’@’localhost’ = PASSWORD(‘root’);
        修改密码成功!
    • 修改mysql 的数据库默认编码方式。
      1. cd /etc
      2. sudo vim my.cnf
        打开后粘贴下面的内容，一会给出。
      3. :wq! –保存后强制退出.
        [附几个编辑命令:dd 删除光标所在行; dw 删除一个字(word); x 删除当前字符].
