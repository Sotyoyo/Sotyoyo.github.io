ubuntu 服务器环境搭建

发表于 2018-06-14

环境搭建

文件上传环境

  1. sudo apt install lrzsz

node环境

  1. 下载linux版本node并且解压

  2. 移动解压后的目录至 /opt/

  3. 测试是否成功

  4. sudo ln -s /opt/node目录名/bin/node /usr/local/bin/node

    sudo ln -s /opt/node目录名/bin/npm /usr/local/bin/npm

npm环境

  1. npm config set registry https://registry.npm.taobao.org

  2. npm get registry

开放端口

  1. 安装firewall-cmd

    sudo apt install firewalld

  2. 基本命令

    firewall-cmd --add-port=8080/tcp --zone=public //添加8080端口

    firewall-cmd --add-port=80/tcp --zone=public //添加80端口

    firewall-cmd —reload //刷新防火墙使设置生效

    firewall-cmd --list-all-zones //查看所有域开放的端口

    firewall-cmd --list-all //查看当前域开放端口
